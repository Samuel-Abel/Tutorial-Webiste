Functions

A function in javascript is a set of code that's supposed to carry out a specific task. It's defined with the 'function' keyword followed by the name of the function and then a set of parentheses used to set parameters for the function. The code that runs inside of a function are placed in between curly brackets. In order for the function to do something it must be called like console logging it or invoked by an event such as a user hovering over something.

Objects

Objects are variables in javascript with different properties. They can be strings, numbers, arrays, etc. The properties of the object come after naming the object and the curly brackets. The properties are key/value pairs, the keys being the property names and the value being anything mentioned above.

Arrays

Arrays in javascript are used to store multiple values in a single variable and to do that you have to define the variable and then follow that with the values inside of regular braces.
